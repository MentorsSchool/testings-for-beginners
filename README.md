## How to run tests
### Setup
You should run the following command when you use RSpec.

```
$ bundle install
```


### minitest

```
$ ruby test/ex<exercise_number>_test.rb
```


### RSpec

```
$ rspec spec/ex<exercise_number>_spec.rb
```
